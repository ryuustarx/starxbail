## ✨ Deskripsi
*RyuuStar-Baileys* adalah Baileys versi modifikasi untuk WhatsApp Web API yang dirancang agar lebih fleksibel, ringan, dan mudah dikembangkan.  
Project ini tetap mempertahankan inti Baileys, namun dengan penyesuaian struktur, optimasi, dan dukungan fitur custom yang cocok untuk bot WhatsApp modern.

## 📦 Installasi
Install langsung lewat *npm*:

```bash
npm i ryuustar/baileys
```
